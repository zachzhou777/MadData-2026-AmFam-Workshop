# MadData 2026 AmFam Workshop
American Family Insurance workshop at UW-Madison's [MadData hackathon](https://dotdatauw.vercel.app/maddata).

## Installation
We use [uv](https://docs.astral.sh/uv/getting-started/installation/) for Python dependency management:
```bash
# Clone the repository
git clone https://github.com/zachzhou777/MadData-2026-AmFam-Workshop.git
cd MadData-2026-AmFam-Workshop

# Create a virtual environment and install dependencies
uv sync
```
This will create a virtual environment and install all dependencies specified in `pyproject.toml`.
